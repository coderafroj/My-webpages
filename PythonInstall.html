Python ek bahut hi lokpriya programming language hai. Is guide mein aapko bataya jayega ki kaise aap apne system par Python install kar sakte hain. Yeh guide Windows, macOS, Linux, aur Android sabhi platforms ke liye hai.

Windows Installation

Windows par Python install karne ke liye neeche diye gaye steps follow karein:

  • Step 1: Python ki official website python.org/downloads par jayein.
  • Step 2: Windows installer (executable) download karein.
  • Step 3: Installer run karte waqt "Add Python to PATH" option ko select karein.
  • Step 4: Installation complete hone ke baad Command Prompt kholkar python --version command run karein.

macOS Installation

macOS par Python install karne ke liye neeche diye gaye steps follow karein:

  • Step 1: Python ki official website se macOS installer download karein.
  • Step 2: Installer package (.pkg file) ko open karke instructions follow karein.
  • Step 3: Terminal kholkar python3 --version command run karein (kyunki macOS mein Python 2 bhi available hota hai, isliye python3 ka upyog karein).

Linux Installation

Linux distributions par Python install karne ke liye terminal commands ka upyog karein:

  • Debian/Ubuntu: 
    sudo apt update
sudo apt install python3
  • Fedora: 
    sudo dnf install python3
  • Arch Linux: 
    sudo pacman -S python

Android Installation

Android device par Python run karne ke liye do popular apps ka upyog kiya ja sakta hai:

  • Pydroid 3: Google Play Store se download karein. Yeh app ek complete Python IDE provide karta hai jisme aap directly Python scripts run kar sakte hain.
  • Termux: Termux app install karein, phir terminal mein pkg install python command run karein.

In apps ke through aap Python scripts ko directly apne Android device par execute kar sakte hain.
