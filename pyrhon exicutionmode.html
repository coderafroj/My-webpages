Python Execution Modes

Python ke do main execution modes hote hain:

  • IDLE (Interactive Mode): Python ke saath aata hai, jismein hum ek-ek line likh ke turant execute kar sakte hain. 
    >>> print("Hello, Python!")

    Output:

    Hello, Python!
  • Script Mode: Python script (.py file) likh ke complete programs run kar sakte hain.

    Example: Python First Program

    # Pehla Python program
print("Welcome to Python!")

    Is file ko filename.py naam se save karein aur terminal/command prompt mein run karein:

    python filename.py

String vs String Arithmetic Operations

Python mein strings sirf + operator ke saath concatenate hote hain:

print("Hello" + " World")

Output:

Hello World

Lekin agar hum -, *, ya / operators ka use karein toh error aayega:

print("Hello" - "World")  # Error
print("Hello" * "World")  # Error
print("Hello" / "World")  # Error

String vs Integer Arithmetic Operations

String aur integer ke beech + operator se error aayega:

print("Hello" + 5)  # TypeError

Agar multiply (*) karein toh string repeat ho sakti hai:

print("Hello" * 3)

Output:

HelloHelloHello

Lekin / ya - operators ka use karne par error aayega:

print("Hello" - 3)  # Error
print("Hello" / 3)  # Error
