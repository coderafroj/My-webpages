Variables & Data Types

Parichay

Python ke programming model ka mool aadhaar hai variables aur unke sath associated data types. Is guide mein hum advanced level par numbers, strings, booleans, aur type casting ko deeply samjhenge, taaki aap apne code ko robust aur efficient bana sakein.

Variables & Dynamic Typing

Python ek dynamically typed language hai. Iska matlab hai ki aapko variable ka type explicitly declare karne ki zaroorat nahi; variable ko assigned value ke hisaab se hi type decide ho jata hai. 

# Dynamic typing example
x = 10          # x is an integer
x = "Hello"     # Now x becomes a string

Numbers

Python mein numbers alag-alag types ke hote hain:

  • Integers: Purnank, jaise 10, -5, 0.
  • Floating-point: Decimal numbers, jaise 3.14, -0.001.
  • Complex Numbers: Jisme real aur imaginary parts hote hain, jaise 2 + 3j.
# Number examples
a = 15           # Integer
b = 3.14         # Float
c = 2 + 3j       # Complex number

# Basic arithmetic
sum = a + b
product = a * b

Strings

Strings text ko represent karte hain. Python mein strings immutable hoti hain, matlab ek baar create ho gayi to usme direct changes nahi kiye ja sakte. Aap concatenation, slicing, aur formatting se strings ko manipulate kar sakte hain. 

# String examples
s1 = "Hello"
s2 = "World"
s3 = s1 + " " + s2       # Concatenation
print(s3)               # Output: Hello World

# Slicing
print(s3[0:5])          # Output: Hello

# Formatting using f-string
name = "Alice"
greeting = f"Welcome, {name}!"
print(greeting)

Booleans

Boolean data type keval do hi values leta hai: True aur False. Ye logical decisions, comparisons, aur control flow ke liye bahut important hote hain. 

# Boolean examples
is_valid = True
is_empty = False

# Using booleans in conditions
if is_valid:
    print("Condition is True")

Data Structures: List, Tuple, Dictionary, Set

Python mein built-in data structures aapko complex data ko store aur manipulate karne mein madad karte hain.

  • List: Ordered, mutable sequence of items. 
    # List example
my_list = [1, 2, 3, "Python", True]
print(my_list)
  • Tuple: Ordered, immutable sequence of items. 
    # Tuple example
my_tuple = (1, 2, 3, "Python", True)
print(my_tuple)
  • Dictionary: Unordered collection of key-value pairs. 
    # Dictionary example
my_dict = {"name": "Alice", "age": 30, "city": "Delhi"}
print(my_dict)
  • Set: Unordered collection of unique items. 
    # Set example
my_set = {1, 2, 3, 3, 2}
print(my_set)  # Output: {1, 2, 3}

Type Casting

Type casting se aap ek data type ko doosre data type mein convert kar sakte hain. Python mein built-in functions jaise int(), float(), str(), aur bool() ka istemal hota hai. 

# Converting a string to an integer and float
num_str = "100"
num_int = int(num_str)       # Converts to integer
num_float = float(num_str)   # Converts to float

# Converting numbers to a string
text = str(num_int)

# Boolean conversion
val1 = bool(0)      # False
val2 = bool(5)      # True

Advanced: Type Hints

Python 3.5 se, type hints se aap apne code ke function parameters aur return types ko explicitly define kar sakte hain, jisse code ki readability aur debugging asaan ho jati hai. 

# Type hints example
def add_numbers(a: int, b: int) -> int:
    return a + b

result: int = add_numbers(10, 20)
print(result)  # Output: 30

Conclusion

Variables aur data types Python ke programming model ka mool hissa hain. Inhein samajhne se aap complex data manipulations aur logical operations ko effectively implement kar sakte hain. Regular practice aur experimentation se aap in advanced concepts mein proficiency hasil kar sakte hain.
