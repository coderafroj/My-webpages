Python Syntax Guide

Parichay

Python syntax ka mool uddeshya code ko simple, readable aur efficient banana hai. Is guide mein hum Python ke basic structure, indentation, statements, aur advanced syntax features ko deeply cover karenge.

Basic Structure

Python programs sequentially execute hote hain. Har program mein statements ek certain order mein likhe jate hain. Yeh structure code ko asaani se follow karne layak banata hai. 

# Ek simple Python program
print("Hello, World!")

Indentation

Python mein indentation ka bahut mahatva hai. Code blocks ko define karne ke liye whitespace ka upyog kiya jata hai. Galat indentation se IndentationError aata hai. 

if 5 > 2:
    print("5 is greater than 2")  # Sahi indentation
else:
    print("This will not print")

Statements & Expressions

Python code ka mool tatva statements hote hain. Har statement ek action ko represent karta hai. Expression se calculation ya logical operation perform hota hai. 

# Assignment statement
x = 10

# Expression statement
x + 5

# Conditional statement
if x > 5:
    print("x is greater than 5")

Variables & Data Types

Python dynamically typed language hai jahan variable declare karne ke liye value assign karna hi kaafi hai. Alag-alag data types, jaise integer, float, string, list, etc., available hain. 

# Variable assignments
num = 42            # Integer
pi = 3.14159        # Floating-point
greeting = "Hello"  # String

Functions & Modules

Functions code ko reusable blocks mein todte hain. def keyword ka use karke function banaya jata hai, jisse aap repetitive tasks ko modular tarike se perform kar sakte hain. 

def add(a, b):
    """Do numbers ka sum return karta hai."""
    return a + b

print(add(5, 3))  # Output: 8

Comments & Documentation

Code ko samjhane ke liye comments ka upyog kiya jata hai. Ek-line comments ke liye # aur multi-line comments ke liye triple quotes (''' ... ''' ya """ ... """) ka use hota hai. 

# Yeh ek single-line comment hai

"""
Yeh ek multi-line comment hai.
Iska upyog documentation ke liye hota hai.
"""

Advanced Syntax Features

Python mein list comprehensions, lambda functions, generator expressions jaise advanced features available hain, jo concise aur expressive code likhne mein madad karte hain. 

# List Comprehension Example
squares = [x*x for x in range(1, 11)]
print(squares)

# Lambda Function Example
multiply = lambda a, b: a * b
print(multiply(3, 4))

Conclusion

Python syntax ko samajhna code likhne ki kala ka mool hai. Is guide mein basic structure, indentation, statements, variables, functions aur advanced features cover kiye gaye hain. In concepts ki practice se hi aap ek efficient programmer ban sakte hain.
